Definition: 
 1: System Design is the process of defining the architecture, components, modules, interfaces, and data 
flow of a system to meet specific requirements like scalability, performance, reliability, 
and maintainability. 

 2: How you plan and structure a complete software system before building it.

 3: System design is about designing systems that scale efficiently and handle real-world traffic.

 type of system design:
 1:  HLD (High-Level Design)
 2:  LLD (Low-Level Design)

  1:  HLD (High-Level Design)
    Overall system architecture

  2: LLD (Low-Level Design)  
   Detailed implementation
   Classes, functions, APIs, schemas
